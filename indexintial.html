HURDLE GAME




TO START THE GAME CLICK ON THE BELOW ICON



Instruction


The game starts, the user has to dodge/hurdle the approaching obstacle
but the user will be at that position only for every successful hurdle
the user gets a +1 point and at last your score will be displayed.
If you collide with the obstacle the game will be over and you will
see your score otherwise the game will keep ongoing for an infinite time.
